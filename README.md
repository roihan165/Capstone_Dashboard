*NOTE: jalankan file bernama main.py yang berada didalam folder backup, untuk file app.py itu berbeda dan menggunakan flask*

Prasyarat Kebutuhan:
Python ver 3.11

Cara Menjalankan:

1. Konfigurasi Enviroment
- Buat virtual env di lokal
```
python -m venv *nama_folder_env*
```
- Gunakan env yang sudah dibuat
jika menggunakan bash
```
*nama_folder_env*\Scripts\activate
```
bila menggunakan yang lain silahkan cari sendiri hehehe
- Install Package
pastikan berada di folder yang terdapat requirements.txt
```
pip install -r requirements.txt
```
- Masuk ke dalam folder backup menggunakan CD command

2. Menjalankan Streamlit
- jalankan file
```
streamlit run main.py
```
